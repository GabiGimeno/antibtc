# AntiBTC (ANTI)

AntiBTC es un token experimental desplegado en la red Base que explora una idea simple:

> ¿Qué ocurre cuando la escasez existe desde el primer bloque y todo el suministro está disponible desde el primer día?

No es una alternativa a Bitcoin ni pretende competir con él.  
AntiBTC es un experimento conceptual sobre escasez, mercado y distribución.

---

## Filosofía

Bitcoin nació como una alternativa al sistema financiero tradicional: descentralizado, sin intermediarios y resistente a la manipulación.

Con el tiempo, una parte significativa del suministro de Bitcoin ha sido absorbida por instituciones, fondos y grandes actores financieros, alterando el contexto de su narrativa original.

AntiBTC nace como respuesta conceptual a ese escenario.

No propone reemplazar a Bitcoin, sino observar qué sucede cuando
